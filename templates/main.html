A proxy server is used to process and forward your HTTP requests to their respective destinations. In order to set up your browser to use map.me - complete the following steps:

  1. Navigate to the following link and find your browser or operating system
  2. Select 🔏 Get mitmproxy-ca-cert.pem and download the certificate
  3. Select 📖 Show Instructions and follow the instructions to install the certificate
